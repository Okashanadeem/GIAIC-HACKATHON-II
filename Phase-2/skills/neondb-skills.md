# Neon DB Skills

This document outlines the key skills and knowledge required for using Neon DB in Phase 2 of this project.

## Core Concepts

*   **Serverless Postgres:** Understanding the architecture and benefits of a serverless PostgreSQL database like Neon.
*   **Project Setup:** Creating a new Neon project and getting the connection string.
*   **Database Branching:** Using database branching to create isolated environments for development, testing, and production.
*   **Connection Pooling:** Understanding how Neon handles connection pooling and how to connect from serverless functions.
*   **Integration:** Connecting to Neon from backend applications (e.g., FastAPI) using standard PostgreSQL drivers.

## Advanced Concepts

*   **Autoscaling:** Understanding how Neon's autoscaling works and how it affects performance and cost.
*   **Point-in-Time Recovery:** Using Neon's point-in-time recovery feature to restore the database to a previous state.
*   **Monitoring and Logging:** Using Neon's dashboard to monitor database performance and view logs.
